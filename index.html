Welcome

This page serves as an index for my 20 Days | 20 Technologies Internship Challenge. Each day focuses on one networking or cybersecurity technology, with practical configurations and documentation to track my progress.

Challenge Days

  • Day 01 — VMware ESXi install + Windows VM + RDP: Students create their first Windows VM, access via RDP.
  • Day 02 — Ubuntu latest VM + SSH remote from India→Paris: Basic Linux provisioning + remote shell over WAN.
  • Day 03 — ESXi Networking + Cisco IOS router: Each student VM placed in unique subnet; NAT + reachability.
  • Day 04 — pfSense firewall VM: Setup WAN/LAN, static routes, and NAT for all student VMs.
  • Day 05 — WireGuard VPN: Secure tunnel between Paris DC and India clients.
  • Day 06 — OpenVPN: Site-to-site & client mode; compare with WireGuard.
  • Day 07 — Tailscale Mesh VPN: Zero-config mesh across all students; subnet routing.
  • Day 08 — FreeRADIUS + Captive Portal: Authentication for local users through pfSense.
  • Day 09 — DNS/DHCP Server: Central DHCP scopes, custom internal DNS zones.
  • Day 10 — Migrate to Proxmox: Import VMs, configure virtual bridge & VLAN-aware networking.
  • Day 11 — MikroTik RouterOS: Create VLANs, NAT, static/dynamic routing; remote management.
  • Day 12 — OSPF with Cisco + MikroTik: Dynamic routing in hybrid lab. Verify LSAs & routes.
  • Day 13 — FRRouting (Linux Router): Replace Cisco with open-source router; redistribute routes.
  • Day 14 — IDS/IPS – Suricata / Zeek: Deploy network monitoring + traffic inspection.
  • Day 15 — Zabbix / Grafana / Prometheus: Centralized monitoring for ESXi, Proxmox, routers, VPNs.
  • Day 16 — Ansible Automation: Automate VM config, IP setup, and user creation.
  • Day 17 — Docker Networking Basics: Deploy small web app + reverse proxy to tie infra & dev.
  • Day 18 — Proxmox Cluster + HA setup: Multi-node cluster; migrate VM live between nodes.
  • Day 19 — Backups, Snapshots & Restore: Run complete backup policy; disaster-recovery drill.
  • Day 20 — Final Integration Project: One cohesive infra: Proxmox + pfSense + MikroTik + Cisco + VPN + Monitoring. Students present design diagrams + live demo.
